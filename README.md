# Portfolio - macOS-Inspired Interactive Portfolio

A modern, interactive portfolio website built with React that emulates the macOS user interface. Features draggable windows, a dock navigation system, and dynamic content showcasing projects, skills, and experience.

## Features

- **macOS-Style Interface**: Authentic macOS UI with draggable windows and a functional dock
- **Interactive Windows**: Multiple window types including Finder, Safari, Terminal, Resume, and Contact
- **Project Showcase**: Navigate through projects in a Finder-like interface with folders and files
- **Skills Terminal**: Terminal-style display of tech stack and skills
- **Articles Section**: Safari-inspired browser for blog posts and articles
- **Responsive Design**: Fully responsive across devices
- **Smooth Animations**: GSAP-powered animations for enhanced user experience

## Tech Stack

- **Frontend**: React 19, React Router
- **Build Tool**: Vite
- **Styling**: Tailwind CSS 4
- **Animations**: GSAP with Draggable plugin
- **State Management**: Zustand
- **Icons**: Lucide React
- **PDF Viewer**: react-pdf
- **Date Handling**: Day.js
